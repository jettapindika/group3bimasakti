# group3bimasakti

🏎️ **Bimasakti FSAE Racing Team Website** - Official website showcasing our Formula SAE race car with premium animations and interactive features.

## ✨ Features

- 🎨 **Hover Reveal Effect** - Interactive circular spotlight reveals car details
- 🎵 **Audio Player** - Background music player with controls
- ✨ **Smooth Animations** - Framer Motion powered animations
- 📱 **Responsive Design** - Optimized for all devices
- 🎯 **Custom Cursor** - 8-particle trail effect
- 🎨 **Orange/Brown Theme** - Racing-inspired color palette

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Visit `http://localhost:3000`

- **Framework**: Next.js 16.0.1
- **UI**: React 19 + Tailwind CSS
- **Animations**: Framer Motion
- **Components**: Shadcn/ui
- **Typography**: Poppins font
